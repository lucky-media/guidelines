This guide covers how we work at [Lucky Media](https://www.luckymedia.dev). It includes our tools, practices, and team culture. It's written for developers.

We build digital products using a specific tech stack and company culture.

## What's in This Guide

- **Roadmaps** - Tutorials for frontend and backend technologies.
  - Frontend Roadmap
  - Backend Roadmap
  - Design Roadmap
- **Editor Configuration** - VS Code settings and extensions we use.
- **Version Control** - How we manage code.
  - Git - Core git usage and workflows.
  - GitHub - How we use GitHub for collaboration.
- **Frontend Technologies** - Frontend tools and how we use them.
  - Tailwind - TailwindCSS setup and plugins.
  - Alpine.js - How we use Alpine.js in projects.
  - React.js - React usage, including our Next.js Starter Kit.
- **Backend Technologies** - Backend tools and how we use them.
  - Laravel - Laravel setup and best practices.
- **Guides** - Step-by-step instructions for specific tasks.
  - Statamic - How to upgrade from Statamic v2 to v3.
- **Debugging** - How to find and fix bugs.
- **Testing Strategies** - How we test our apps.
- **Code Review Best Practices** - How we review code.
- **Deployment Guidelines** - How we deploy projects.
- **Company Culture** - How our team is structured, how we work, and what we value.
